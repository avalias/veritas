The research · the breakthrough

The VM is the math.

You don’t need a CPU emulator to run a neural net. A transformer’s forward pass is just ~27 tensor micro-ops — a block dot product FDOT, scalar float ops FOP, and a few layout shuffles.

So the referee doesn’t interpret an instruction set — it replays one micro-op. A dispute over one judgment (≈ 30M float micro-ops) bisects in ~25 rounds down to a single on-chain verify_step at chain-minimum gas.

opml/compiler turns any quantized LLM with float activations into a VM program plus genesis pages — a compiler and a Move package.

dispute::verify_step · move
// the whole referee, one micro-op at a time
public fun verify_step(op, page, claim): Root {
if (op == FDOT) block_dot(page) // 1 of ~27
else if (op == FOP) scalar_op(page)
else shuffle(page)
}
// 30M ops → ~25 rounds → ONE verify_step
// referee total: ~330 lines of Move
