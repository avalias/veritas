Why one input gives one answer

This model is committed-float: every arithmetic step is pinned to a fixed rounding, so the same evidence produces the bit-identical output on any CPU or GPU. The published checkpoint reads at perplexity 34.60; that number is the fingerprint of the exact weights running here.

Determinism is what makes a verdict disputable: there is one right output, so a resolver who reports a different one can be caught. The judge only reads the evidence. The market's committed counting rule decides the outcome, never the AI's opinion.
